# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `@anivar/mobile-ai-toolkit`, please report it privately via GitHub's [private vulnerability reporting](https://github.com/openslm-ai/mobile-ai-toolkit/security/advisories/new).

Please do not open a public issue for security-sensitive reports.

You can expect an initial response within 7 days. Fixes for confirmed vulnerabilities will be released as soon as practical, with a coordinated disclosure timeline.

## Supported Versions

Only the latest minor release line receives security updates.
